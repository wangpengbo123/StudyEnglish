一、毕业设计（论文）的主要内容（含主要技术参数）
1主要内容：
英语四、六级考试作为高校英语教学效果的评价指标,对于理工科学生同样具有重要的意义。针对智能环境进行基于Java四六级小助手APP的设计与开发，可以为学生提供更多便利，促使学生利用碎片化时间进行英语学习，从而提高考试成绩。
系统包含手机客户端与 WEB 服务器管理端，系统设计应具备两大功能，即学习与测试。本次毕业设计主要的内容包括:
1)	手机客户端主要针对学生，确保学生能够不限时间地点地学生，具备语法学习、测试、词典查询、留言等功能。
2)	2WEB服务器端针对的是管理人员，管理人员基于 PC 机端维护系统，并管理试题、处理留言。采用MVC框架实现系统功能。
3)	学生通过注册登录后选择想要测试的内容进行在线答题，在规定的时间内完成所有题目，否则时间一到系统自动提交试卷，提交试卷后，系统进行评分并反馈给用户。
4)	试卷导入建议使用JDBC，实现对数据库的访问。
2. 主要技术参数
本系统总体上采用C/S体系结构，服务器端管理平台采用B/S体系结构。
开发环境：客户端采用 Android Studio或者eclipse+androidSDK，服务器端采用MyEclipse和Tomcat；
操作系统：采用windows 7/8/10系统；
数据库：客户端采用SQLite，服务器端采用MySql；
开发语言：使用Java语言进行客户端和服务器端的开发。
